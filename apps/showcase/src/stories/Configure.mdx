## 🚀 Inicio Rápido en 3 Pasos

<div class="pipeline-step">
  <h4>📦 Paso 1: Instalar Dependencias</h4>
  <p>Añade las dependencias necesarias a tu proyecto React.</p>
</div>

```bash
# NPM
npm install @liquid-svg-glass/core @liquid-svg-glass/react

# Yarn
yarn add @liquid-svg-glass/core @liquid-svg-glass/react

# PNPM  
pnpm add @liquid-svg-glass/core @liquid-svg-glass/react

# Bun (recommended)
bun add @liquid-svg-glass/core @liquid-svg-glass/react
```

<div class="tip-box">
  <strong>💡 Nota:</strong> GSAP se incluye automáticamente con @liquid-svg-glass/react para funcionalidad de arrastre. React 19+ es requerido.
</div>

<div class="pipeline-step">
  <h4>📥 Paso 2: Importar Componente</h4>
  <p>Importa el componente GlassEffect en tu aplicación React.</p>
</div>

```tsx
import { GlassEffect } from '@liquid-svg-glass/react';

// También puedes importar tipos y presets
import { GlassEffect, glassPresets, type GlassEffectProps } from '@liquid-svg-glass/react';
```

<div class="pipeline-step">
  <h4>✨ Paso 3: Usar el Efecto</h4>
  <p>Añade el efecto de cristal a cualquier contenido con un simple preset.</p>
</div>

```tsx
<GlassEffect preset="dock">
  <nav>Tu contenido aquí</nav>
</GlassEffect>
```

<div class="info-box">
  <strong>🏗️ Arquitectura Monorepo:</strong> Este proyecto utiliza un monorepo con dos paquetes principales:
  <ul>
    <li><strong>@liquid-svg-glass/core</strong> - Utilidades y tipos framework-agnostic</li>
    <li><strong>@liquid-svg-glass/react</strong> - Componentes React que usan el core</li>
  </ul>
</div>

<div class="success-box">
  <strong>🎉 ¡Listo!</strong> Ya tienes un efecto de cristal líquido funcionando. El preset "dock" es perfecto para barras de navegación.
</div>

## 📋 Ejemplos Básicos

### Efecto de Cristal Estático (Barra de Navegación)

```tsx
<GlassEffect draggable={false} preset="dock">
  <nav className="main-nav">
    <a href="/">Inicio</a>
    <a href="/acerca">Acerca</a>
    <a href="/contacto">Contacto</a>
  </nav>
</GlassEffect>
```

### Elemento Arrastrable Interactivo

```tsx
<GlassEffect 
  draggable={true}
  preset="pill"
  initialPosition={{ x: 100, y: 50 }}
  onMove={(pos) => console.log('Movido a:', pos)}
>
  <div className="widget">
    <h3>Widget Arrastrable</h3>
    <p>¡Arrástrame por la pantalla!</p>
  </div>
</GlassEffect>
```

### Efecto de Cristal con Tamaño Automático

```tsx
<GlassEffect autoSize preset="dock">
  <button className="glass-button">
    Botón con efecto de cristal auto-ajustable
  </button>
</GlassEffect>
```

### Configuración Personalizada

```tsx
<GlassEffect
  preset="free"
  width={400}
  height={200}
  blur={15}
  r={60} g={70} b={50}
  blend="multiply"
  debug={true}
>
  <div className="custom-content">
    <h2>Efecto de Cristal Personalizado</h2>
    <p>Parámetros totalmente personalizados</p>
  </div>
</GlassEffect>
```

Habilita el modo debug durante el desarrollo:


El modo debug muestra:

## 🔧 Troubleshooting Común

<details class="collapsible">
  <summary><strong>❓ Problemas Frecuentes y Soluciones</strong></summary>
  <div class="collapsible-content">

### El efecto no se ve o es muy sutil

<div class="warning-box">
  <strong>Causa:</strong> Falta soporte de <code>backdrop-filter</code> o no hay contenido detrás del elemento.
</div>

**Soluciones:**
<ul>
<li>Verifica que tu navegador soporte <code>backdrop-filter</code></li>
<li>Asegúrate de que hay contenido visible detrás del elemento</li>
<li>Aumenta el valor de <code>blur</code> para un efecto más pronunciado</li>
</ul>

### Performance degradada en móviles

<div class="tip-box">
  <strong>Solución:</strong> Reduce la complejidad del filtro en dispositivos móviles.
</div>

```tsx
<GlassEffect 
  preset="dock"
  blur={isMobile ? 5 : 11}
  r={isMobile ? 25 : 50}
  g={isMobile ? 30 : 55}
  b={isMobile ? 20 : 45}
>
  Contenido optimizado para móvil
</GlassEffect>
```

### El contenido no es arrastrable

**Verifica:**
<ul>
<li>GSAP está instalado: <code>npm install gsap</code></li>
<li>La prop <code>draggable={true}</code> está configurada</li>
<li>No hay overlays CSS que bloqueen el input</li>
</ul>

  </div>
</details>
